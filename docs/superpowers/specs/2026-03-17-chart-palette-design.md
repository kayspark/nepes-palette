# Nepes Chart Palette вЂ” Design Spec

## Overview

Create chart color palettes for R (ggplot2) and Python (matplotlib) derived from the nepes corporate colorscheme. Light theme is default. Two new repos: `ggnepes` (R/CRAN) and `mplstyle-nepes` (Python).

**API:**

```r
library(ggnepes)

# Palette access
nepes_pal("light")           # 12-color character vector
nepes_pal("light", n = 6)    # first 6 colors
nepes_pal("dark")             # dark variant
nepes_pal()                   # default = "light"

# ggplot2 scales (discrete)
scale_color_nepes(theme = "light")   # default
scale_fill_nepes(theme = "dark")

# ggplot2 themes (optional)
theme_nepes_light()    # white bg, styled axes/grid/text
theme_nepes_dark()     # #1E1C1A bg, light text
```

**Color values (light, 12 series):**

From `palette_data.py` light theme, `chart_series()` with 6 bases + 6 lightened:
- Bases: `#23438E`, `#D08A10`, `#2A8030`, `#7A4FA0`, `#2D7A82`, `#C4181F`
- Extended: lightened variants via `chart_series(bases, 12)`

**Color values (dark, 12 series):**
- Bases: `#5C8CFF`, `#FEA413`, `#3DDC84`, `#A274C3`, `#3A9BA5`, `#FF5C5C`
- Extended: lightened variants

**Theme functions:**

`theme_nepes_light()`:
- White background, light gray grid lines
- Font: system sans-serif (no custom font dependency)
- Axis text/title colors from light palette fg ramp
- No border, clean minimal style

`theme_nepes_dark()`:
- Background `#1E1C1A`, panel `#1E1C1A`
- Grid lines `#3E3A38`, text `#DCD8D4`
- Same minimal structure

**CRAN requirements:**
- No compiled code (pure R)
- roxygen2 documentation for all exported functions
- testthat tests
- Examples in man pages
- DESCRIPTION: Title, Description, Authors@R, License, Imports

### 3. Generators in `nepes-palette`

Two new generators:
- `generators/ggplot2.py` вЂ” produces `R/palettes.R` with hex vectors
- `generators/mplstyle.py` вЂ” produces `.mplstyle` files

Registered in `generators/__init__.py` alongside existing 16 generators.

## Generation Flow

```
palette_data.py  в†’  generators/ggplot2.py   в†’  ggnepes/R/palettes.R
                 в†’  generators/mplstyle.py  в†’  mplstyle-nepes/nepes-{dark,light}.mplstyle
```

The R package and mplstyle files contain hardcoded hex values (no runtime dependency on Python). Regenerate when palette changes.

## Out of Scope

- PyPI package (just .mplstyle for now)
- Domain-specific palettes (SPC, sequential, diverging)
- Continuous color scales (only discrete)
- Font bundling
- plotly / seaborn-specific integrations
